# CreditDB

Engine for [CreditDB](https://github.com/creditdb), a fast and efficient key-value database built in Typescript.

## Installation

Clone the repo from github:

```sh
git clone https://github.com/creditdb/creditdb.git
```

Install dependencies:

```sh
cd creditdb
npm install
```

## Usage

Run the application:
```sh
npm start
```
